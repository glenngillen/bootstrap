# OSX Bootstrap

Builds a dev machine for me. Based on [Ben Bleything's Bootstrap](https://github.com/bleything/bootstrap).

## Usage

    bash < <(curl -s https://raw.github.com/glenngillen/bootstrap/master/bin/bootstrap)
